Calculator

Rules and Hints :

  • Calculator has autofocus
  • Enter only numbers
  • You can either type numbers or press the buttons
  • Entering character will cause an error
  • After pressing "=" button the result will apear
  • You cannot continue computation with "=" in the input
    • (You have to manually delete it and continue)
  • Sqrt and Power has its own executer: That means to get a result you have to press them, not the "=" buton
  • Ex:
    • input = 4 -> press sqrt(not the "=") -> get a result of 2
  • To get a power of a number you must provide 2 numbers: Base and Power. To Separate them you must enter power in the input field
  • Ex:
    • input = 4power2 -> press power(not the "=") -> get a result of 16
    • input = 2+3*4power4-2 -> equal to 2+(3*4)^(4-2) -> press power(not the "=") -> get a result of 196
